## Введение

Движение тела, брошенного под углом к горизонту, является одной из фундаментальных задач классической механики и теоретической баллистики [1]. Эта задача рассматривается в основных курсах физики и теоретической механики [2] и служит основой для более сложных моделей, учитывающих сопротивление среды, вращение Земли и другие факторы [3].

Целью данной работы является построение и анализ математической модели движения камня, брошенного под углом к горизонту, с последующей визуализацией траектории и исследованием влияния параметров броска на характеристики полета.

## 1. Математическая модель

Рассмотрим движение материальной точки (камня) массой *m*, брошенной с начальной скоростью \( \vec{v_0} \), направленной под углом \( \alpha \) к горизонту. Систему координат выберем таким образом, что ось *OX* направлена горизонтально, ось *OY* – вертикально вверх, а начало координат совпадает с точкой бросания. Пренебрегаем сопротивлением воздуха и кривизной земной поверхности.

Единственной силой, действующей на тело после броска, является сила тяжести \( m\vec{g} \), направленная вертикально вниз. Запишем уравнения движения согласно второму закону Ньютона в проекциях на оси координат:

\[
m a_x = 0
\]
\[
m a_y = -m g
\]

где \( a_x \) и \( a_y \) – проекции ускорения на оси координат, \( g \approx 9.81 \) м/с² – ускорение свободного падения.

Учитывая, что ускорение есть производная скорости, а скорость – производная координаты, и зная начальные условия (\( x_0 = 0, y_0 = 0, v_{0x} = v_0 \cos \alpha, v_{0y} = v_0 \sin \alpha \)), проинтегрируем уравнения движения.

Скорость тела в произвольный момент времени *t*:
\[
v_x(t) = v_0 \cos \alpha
\]
\[
v_y(t) = v_0 \sin \alpha - g t
\]

Координаты тела в произвольный момент времени *t*:
\[
x(t) = v_0 \cos \alpha \cdot t
\]
\[
y(t) = v_0 \sin \alpha \cdot t - \frac{g t^2}{2}
\quad (1)
\]

Исключив из системы (1) время *t*, получим уравнение траектории – зависимость *y(x)*:
\[
y(x) = x \cdot \tan \alpha - \frac{g x^2}{2 v_0^2 \cos^2 \alpha}
\quad (2)
\]

Уравнение (2) представляет собой параболу, ветви которой направлены вниз.

## 2. Основные характеристики движения

Используя полученные уравнения, определим ключевые параметры полета [2].

1. **Время полета \( t_{\text{пол}} \)** – время от момента броска до падения на землю. В момент падения \( y(t_{\text{пол}}) = 0 \). Из уравнения (1) для *y(t)*, исключая тривиальное решение *t=0*, получаем:
    \[
    t_{\text{пол}} = \frac{2 v_0 \sin \alpha}{g}
    \]

2. **Дальность полета \( L \)** – расстояние от точки бросания до точки падения. Подставим \( t_{\text{пол}} \) в уравнение для *x(t)*:
    \[
    L = x(t_{\text{пол}}) = \frac{v_0^2 \sin 2\alpha}{g}
    \quad (3)
    \]

3. **Максимальная высота подъема \( H \)** – наибольшая координата *y* по достижении которой тело начинает двигаться вниз. В вершине траектории вертикальная составляющая скорости равна нулю: \( v_y(t_{\text{под}}) = 0 \). Отсюда \( t_{\text{под}} = \frac{v_0 \sin \alpha}{g} \). Подставив это время в уравнение для *y(t)*, получим:
    \[
    H = y(t_{\text{под}}) = \frac{v_0^2 \sin^2 \alpha}{2g}
    \]

## 3. Численное моделирование и обсуждение результатов

Для визуализации траектории и проведения параметрического анализа была разработана программа на языке Python с использованием библиотек NumPy и Matplotlib [5, 6]. Программа рассчитывает траекторию по уравнению (2) и строит ее график для заданных пользователем значений \( v_0 \) и \( \alpha \).

### Листинг программы:

```python
import numpy as np
import matplotlib.pyplot as plt

# Параметры модели
g = 9.81  # ускорение свободного падения, м/с²

def calculate_trajectory(v0, alpha, x_max=100, num_points=1000):
    """
    Расчет траектории движения тела, брошенного под углом к горизонту
    
    Parameters:
    v0 - начальная скорость, м/с
    alpha - угол бросания, градусы
    x_max - максимальная дальность для расчета, м
    num_points - количество точек для расчета
    
    Returns:
    x, y - массивы координат траектории
    """
    # Перевод угла в радианы
    alpha_rad = np.radians(alpha)
    
    # Расчет дальности полета
    L = (v0**2 * np.sin(2*alpha_rad)) / g
    
    # Создание массива координат x
    x = np.linspace(0, min(L * 1.1, x_max), num_points)
    
    # Расчет координат y по уравнению траектории
    y = x * np.tan(alpha_rad) - (g * x**2) / (2 * v0**2 * np.cos(alpha_rad)**2)
    
    # Убираем точки, где y < 0 (под землей)
    mask = y >= 0
    return x[mask], y[mask]

# Построение траекторий для разных углов при фиксированной скорости
v0_fixed = 20  # м/с
angles = [30, 45, 60, 75]  # градусы

plt.figure(figsize=(10, 6))
for angle in angles:
    x, y = calculate_trajectory(v0_fixed, angle)
    plt.plot(x, y, label=f'α = {angle}°')

plt.title(f'Траектории полета тела при v₀ = {v0_fixed} м/с')
plt.xlabel('Расстояние, м')
plt.ylabel('Высота, м')
plt.grid(True, alpha=0.3)
plt.legend()
plt.axis('equal')
plt.show()

# Построение траекторий для разных скоростей при фиксированном угле
angle_fixed = 45  # градусы
velocities = [10, 15, 20, 25]  # м/с

plt.figure(figsize=(10, 6))
for v in velocities:
    x, y = calculate_trajectory(v, angle_fixed)
    plt.plot(x, y, label=f'v₀ = {v} м/с')

plt.title(f'Траектории полета тела при α = {angle_fixed}°')
plt.xlabel('Расстояние, м')
plt.ylabel('Высота, м')
plt.grid(True, alpha=0.3)
plt.legend()
plt.axis('equal')
plt.show()
