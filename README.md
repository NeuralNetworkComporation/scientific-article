Математическая модель
Рассмотрим движение материальной точки (камня) массой m, брошенной с начальной скоростью V0, направленной под углом α к горизонту. Систему координат выберем таким образом, что ось OX направлена горизонтально, ось OY – вертикально вверх, а начало координат совпадает с точкой бросания. Пренебрегаем сопротивлением воздуха и кривизной земной поверхности.

Единственной силой, действующей на тело после броска, является сила тяжести mg, направленная вертикально вниз. Запишем уравнения движения согласно второму закону Ньютона в проекциях на оси координат:

m·ax = 0
m·ay = -m·g

где ax и ay – проекции ускорения на оси координат, g ≈ 9.81 м/с² – ускорение свободного падения.

Учитывая, что ускорение есть производная скорости, а скорость – производная координаты, и зная начальные условия (x0 = 0, y0 = 0, v0x = v0·cos α, v0y = v0·sin α), проинтегрируем уравнения движения.

Скорость тела в произвольный момент времени t:
vx(t) = v0·cos α
vy(t) = v0·sin α - g·t

Координаты тела в произвольный момент времени t:
x(t) = v0·cos α·t
y(t) = v0·sin α·t - (g·t²)/2 (1)

Исключив из системы (1) время t, получим уравнение траектории – зависимость y(x):
y(x) = x·tan α - (g·x²)/(2·v0²·cos² α) (2)

Уравнение (2) представляет собой параболу, ветви которой направлены вниз.

Основные характеристики движения
Используя полученные уравнения, определим ключевые параметры полета.

Время полета t_пол – время от момента броска до падения на землю. В момент падения y(t_пол) = 0. Из уравнения (1) для y(t), исключая тривиальное решение t=0, получаем:
t_пол = (2·v0·sin α)/g

Дальность полета L – расстояние от точки бросания до точки падения. Подставим t_пол в уравнение для x(t):
L = x(t_пол) = (v0²·sin 2α)/g (3)

Максимальная высота подъема H – наибольшая координата y по достижении которой тело начинает двигаться вниз. В вершине траектории вертикальная составляющая скорости равна нулю: vy(t_под) = 0. Отсюда t_под = (v0·sin α)/g. Подставив это время в уравнение для y(t), получим:
H = y(t_под) = (v0²·sin² α)/(2g)

Численное моделирование и программа
Для визуализации траектории и проведения параметрического анализа была разработана программа на языке Python с использованием библиотек NumPy и Matplotlib.

python
import numpy as np
import matplotlib.pyplot as plt

# Параметры модели
g = 9.81  # ускорение свободного падения, м/с²

def calculate_trajectory(v0, alpha, x_max=100, num_points=1000):
    """
    Расчет траектории движения тела, брошенного под углом к горизонту
    
    Parameters:
    v0 - начальная скорость, м/с
    alpha - угол бросания, градусы
    x_max - максимальная дальность для расчета, м
    num_points - количество точек для расчета
    
    Returns:
    x, y - массивы координат траектории
    """
    # Перевод угла в радианы
    alpha_rad = np.radians(alpha)
    
    # Расчет дальности полета
    L = (v0**2 * np.sin(2*alpha_rad)) / g
    
    # Создание массива координат x
    x = np.linspace(0, min(L * 1.1, x_max), num_points)
    
    # Расчет координат y по уравнению траектории
    y = x * np.tan(alpha_rad) - (g * x**2) / (2 * v0**2 * np.cos(alpha_rad)**2)
    
    # Убираем точки, где y < 0 (под землей)
    mask = y >= 0
    return x[mask], y[mask]

# Построение траекторий для разных углов при фиксированной скорости
v0_fixed = 20  # м/с
angles = [30, 45, 60, 75]  # градусы

plt.figure(figsize=(10, 6))
for angle in angles:
    x, y = calculate_trajectory(v0_fixed, angle)
    plt.plot(x, y, label=f'α = {angle}°')

plt.title(f'Траектории полета тела при v₀ = {v0_fixed} м/с')
plt.xlabel('Расстояние, м')
plt.ylabel('Высота, м')
plt.grid(True, alpha=0.3)
plt.legend()
plt.axis('equal')
plt.show()

# Построение траекторий для разных скоростей при фиксированном угле
angle_fixed = 45  # градусы
velocities = [10, 15, 20, 25]  # м/с

plt.figure(figsize=(10, 6))
for v in velocities:
    x, y = calculate_trajectory(v, angle_fixed)
    plt.plot(x, y, label=f'v₀ = {v} м/с')

plt.title(f'Траектории полета тела при α = {angle_fixed}°')
plt.xlabel('Расстояние, м')
plt.ylabel('Высота, м')
plt.grid(True, alpha=0.3)
plt.legend()
plt.axis('equal')
plt.show()
Результаты и обсуждение
При моделировании были получены следующие результаты:

При фиксированной начальной скорости 20 м/с и различных углах бросания (30°, 45°, 60°, 75°) максимальная дальность полета достигается при угле 45°, что соответствует теоретическим расчетам: sin(2·45°) = sin 90° = 1 - максимальное значение.

С увеличением угла бросания при постоянной начальной скорости наблюдается рост максимальной высоты подъема и времени полета, однако дальность полета сначала увеличивается, достигает максимума при 45°, а затем уменьшается.

При фиксированном угле бросания 45° и различных начальных скоростях (10, 15, 20, 25 м/с) наблюдается квадратичный рост как дальности полета, так и максимальной высоты подъема с увеличением начальной скорости, что полностью соответствует аналитическим выражениям (2) и (3).
